#   光束传播法原理

光束传播法（Beam Propagation Method，BPM）是一种数值模拟技术，用于研究光波在介质中的传播行为。其核心思想是将光场沿传播方向（通常取 z 轴）逐步推进，每一步只考虑当地折射率分布对光场的衍射与相位调制，从而把三维波动方程降维为一系列二维截面的递推问题。

 
## 边界处理
- 透明边界条件（TBC）或完美匹配层（PML）抑制人工反射。
- 横向窗口足够宽，确保能量泄漏可忽略。

## 适用场景
- 光波导（光纤、PLC、硅光芯片）模式演化、耦合与损耗分析。
- 非均匀介质（热透镜、增益/损耗区）中的光束展宽或压缩。
- 器件优化：MMI、AWG、定向耦合器、星形耦合器等。

## 精度与局限
- 旁轴近似限制大角度（>20°）或强反射场景。
- 标量假设忽略偏振耦合；若需矢量效应，可扩展为全矢量 BPM。
- 步长 需满足 以保证数值色散可接受。

通过上述原理，BPM 以较低计算成本获得光场在复杂三维结构中的详细演化信息，成为集成光子学设计与优化的首选工具之一。
